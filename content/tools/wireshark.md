# Wireshark #

Wireshark(前称Ethereal)是一个网络分析软件。Wireshark使用WinPcap作为借口，直接与网卡进行数据报文交换


## Filter ##

### Capture Filter ###

Working Diretory(or Untracted Files) -> Staging Area -> Repository

### Display Filter ###

	1. protocol.field ==,>,<,>=,<=,!=
	2. &&, ||
	3. contains(包含字符串)
