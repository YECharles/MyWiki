破解方法

License server

1. 选择License server
2. License server address中填入地址：http://idea.lanyus.com
3. OK/Activate

Activation code

1. 浏览器中输入地址：http://idea.lanyus.com
2. 点击获取注册码并复制注册码
3. Pycharm注册中选择Activation code
4. 粘贴注册码
5. OK/Activate
